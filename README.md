# LG Aimers 8th

LG Aimers 8기 온라인 해커톤부터 오프라인 해커톤까지의 참가 결과물.

## 진행 경과

| 단계 | 결과 |
|---|---|
| 온라인 해커톤 | 상위 26팀 중 **13위**로 오프라인 진출 |
| 오프라인 해커톤 | 팀 "언발란스"로 참가, **전체 20위**로 마감. 훈련지원금 수령 |
